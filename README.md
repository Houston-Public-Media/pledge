# HPM Pledge Dashboard

This repository contains the code and documentation for the HPM Pledge Dashboard, a web application designed to visualize and analyze donations made to HPM via Springboard and Calls Without Walls.

The dashboard currently consists of two parts: a main dashboard that provides daily, hourly, and day-of-week breakdowns of donations, and an ingest page that allows users to upload CSV files containing donation data.

If you wish to use this, please contact [Jared Counts](mailto:jcounts@houstonpublicmedia.org) for details. You will need to create a .env file and a SQLite database file in the `/app` folder.
